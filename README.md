# eslint-config-belsrc

An opinionated collection of ESLint plugin configs I use in other projects.

* eslint-plugin-fp
* eslint-plugin-import
* eslint-plugin-promise
* eslint-plugin-react
* eslint-plugin-vue

## Install

```bash
npm i -D eslint-config-belsrc
```

## Use

```js
// .eslintrc.js
module.exports = {
  extends: ['eslint-config-belsrc'],
};
```
